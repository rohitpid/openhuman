System Requirements

Linux

The o3d plugin for Linux is required along with a graphics card capable of OpenGL 2.0 or greater. For Ubuntu packages can be found here. For other distributions, you can either use alien to convert the Ubuntu packages or you can compile it with the instructions here.

  • Intel
    • OpenGL 2.1 is available for Intel i965 cards or better with Mesa 7.6 and newer running with kernel 2.6.28 or newer.
  • ATI
    • OpenGL 2.1 currently only available with the proprietary ATI Catalyst drivers for Linux.
    • The open source radeon driver will pick up OpenGL 2.1 support soon.
  • NVIDIA
    • OpenGL 2.1 support is only available with NVIDIA proprietary drivers for Linux.
    • The open source nouveau driver does not currently support OpenGL 2.1.

Windows

The o3d plugin is required along with a graphics card with drivers capable of DirectX 9.0c or greater.

Mac OS

The o3d plugin is required along with a graphics card with drivers capable of OpenGL 2.0 or greater.
